# EasyBase
A Forge utility mod base for 1.12.2

# Requirements
1. Java JDK 8

# Setup
1. Download the repository
2. Run `gradlew clean`
3. Run `gradlew setupDecompWorkspace`
4. Open folder in IDE of choice
